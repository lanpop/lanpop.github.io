# 简介
位运算符重要包括：与（&）、或（|）、异或（^）、取反（~）、左移（<<）、右移（>>）。其中除了取反（~）以外，其他都是双目运算符。  
**补码**，负数是以补码的形式表示的，即将正数的二进制取反，然后加1。
```
-10的二进制，先取10的二进制
0000 0000 0000 1010
取反
1111 1111 1111 0101
加1
1111 1111 1111 0110（-10的二进制值）
```
# &（与运算）
当两个数同位上的数**都是1**时，该位取1，否则取0
```
10&-10=2

0000 0000 0000 1010
1111 1111 1111 0110
-------------------&
0000 0000 0000 0010
```

# |（或运算）
当两个数同位上的数**存在1**时，该位取1，否则取0
```
10|-10=-2

0000 0000 0000 1010
1111 1111 1111 0110
-------------------|
1111 1111 1111 1110
```

# ^（异或运算）
当两个数同位上的数**不同**时，该位取1，否则取0
```
10^-10=-4

0000 0000 0000 1010
1111 1111 1111 0110
-------------------^
1111 1111 1111 1100
```
>可以看出任何数与0异或，结果都是其本身。  
利用异或可以实现一个很好的交换算法，用于交换两个数，算法如下：

```java
int a = 10;
int b = -10;
a = a ^ b;
b = b ^ a;
a = a ^ b;
System.out.println(a);
System.out.println(b);
```
>-10  
10

# ~（取反）
就是取反呗
```
~10=-11

0000 0000 0000 1010
-------------------~
1111 1111 1111 0101
```

# <<（左移）
将一个数的各位上的数向左移n位
```
10<<3=80(将10的各位上的数左移3位)

0000 0000 0000 1010
-------------------
0000 0000 0101 0000
```
>左移n位就相当于原数×2的n次方

# >>（右移）
将一个数的各位上的数向右移n位
```
10>>3=1(将10的各位上的数又移3位)

0000 0000 0000 1010
-------------------
0000 0000 0000 0001
```
>右移n位就相当于原数÷2的n次方，取整。
